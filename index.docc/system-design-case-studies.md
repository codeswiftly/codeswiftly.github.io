Use these case studies to rehearse complete system design stories with diagrams and tradeoffs.

## Case Study Template

- Start with a 45-minute tour script (timeboxed).
- Map the case study to the generic outline:
  - Problem framing
  - Architecture
  - Data flow
  - Caching and offline
  - Scaling
  - Observability
  - Privacy and safety
  - Failure modes
- Link any reusable diagrams or Excalidraw templates.
- End with open questions and next-iteration deltas.
